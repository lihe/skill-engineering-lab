# Sample Product Brief

Product: SkillEval Lab

Goal: Show how an Agent Skill evolves from writing to evaluation to governance.

Visible result: Run the same cases across without_skill, with_skill_v1, and with_skill_v2, then generate a report with trigger recall, pass rate, token cost, badcases, and next actions.

Audience: AI engineers, product builders, and people building internal agent workflows.
