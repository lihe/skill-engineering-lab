Agent Skill Engineering

把 Skill 从提示词，升级成可评测、可治理的工程资产。

这个研究型项目站展示了一个 AI Agent Skill 如何经历编写、评测、失败归因和治理闭环。核心不是“输出一次好结果”，而是用对照实验证明 Skill 是否带来稳定净增益。

Thesis

成熟 Skill 的标准不是“能跑”，而是“可验证地更好”。

本项目把 Skill 评测拆成触发、产出、风格、效率、泛化和治理。每个失败样例都必须回到一个明确动作：改 description、改 SKILL.md、补 references、补 scripts，或加入回归集。
