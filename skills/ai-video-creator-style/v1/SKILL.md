# AI Video Creator Style v1

Turn AI product briefs into video creative ideas.

## Workflow

1. Read the product brief.
2. Generate title candidates.
3. Create cover text.
4. Create a three-part outline.
5. Write a short spoken script.

## Style

- Make it catchy and Bilibili-friendly.
- Mention product value clearly.
- Use a demo when possible.

## Known Weaknesses

- The description is too broad and may trigger on adjacent promotional writing tasks.
- It does not clearly say when not to use this Skill.
- It asks for Bilibili style but does not harden title count, cover line length, or demo-first rules.
- It has no deterministic verifier script.
